# 梅州传统村落地图 - 微信小程序

## 项目简介

这是一个基于微信小程序开发的梅州传统村落地图应用，展示了梅州市内320个传统村落的详细信息，包括建筑形式、历史背景、地形地貌等丰富的文化信息。

## 功能特性

- 🗺️ **地图展示**: 使用微信小程序地图组件展示广东省内的景点位置
- 📍 **标记点**: 不同类型的地点使用不同的图标进行标记
- ℹ️ **信息面板**: 点击标记点可查看地点基本信息
- 📱 **详情页面**: 查看地点的详细信息，包括描述、地址、评分等
- 🖼️ **图片展示**: 支持图片轮播展示（需要真实图片URL）
- 🏷️ **标签分类**: 按地点类型进行分类显示

## 包含的村落类型

1. **横堂屋** - 传统民居建筑
2. **围龙屋** - 客家特色建筑
3. **围屋** - 防御性民居
4. **围寨** - 寨堡式建筑
5. **杠屋** - 客家杠屋建筑
6. **中西合璧** - 近代建筑风格
7. **自由形式** - 其他建筑形式

## 运行要求

- 微信开发者工具
- 微信小程序基础库 2.19.4 或更高版本
- 有效的微信小程序 AppID

## 使用说明

1. 使用微信开发者工具打开项目
2. 在 `project.config.json` 中配置您的 AppID
3. 确保网络连接正常（地图功能需要网络）
4. 点击预览或真机调试

## 注意事项

- 图片链接目前使用的是示例URL，实际使用时需要替换为真实的图片地址
- 地图功能需要用户授权位置权限
- 建议在真机上测试以获得最佳体验
